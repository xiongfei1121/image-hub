# 更新日志

所有重要的更改都将记录在此文件中。

## [2.0.0] - 2026-05-16

### 重大更新

#### 🎨 全新设计

参考 [SlimImg](https://www.slimimg.com/zh) 的设计风格，采用现代简洁的视觉设计：

- **配色方案**: Indigo (#6366f1) 主色调 + 渐变背景
- **组件设计**: 圆角卡片 (rounded-2xl) + 轻微阴影
- **动画效果**: 流畅的过渡和悬停效果
- **响应式设计**: 完美适配移动端和桌面端

#### 🎯 用户体验优化

- **首页**: 新增 Hero 区域 + 功能卡片网格 + 使用步骤说明
- **导航**: 固定顶部导航栏 + 移动端汉堡菜单
- **页脚**: 深色主题页脚 + 快速链接
- **状态提示**: 优化的成功/错误消息样式

### 详细改进

#### 首页 (index.vue)

- ✨ 新增 Hero 区域，包含标题、描述和 CTA 按钮
- ✨ 新增特性介绍区域（极速处理、隐私优先、永久免费）
- ✨ 重新设计功能卡片，添加悬停动画效果
- ✨ 新增使用步骤说明（选择工具 → 上传图片 → 下载保存）
- ✨ 新增 CTA 区域，鼓励用户开始使用

#### 压缩页面 (compress.vue)

- 🎨 页面标题区域：大标题 + 描述文字
- 🎨 设置面板：使用 card 类，添加图标
- 🎨 上传区域：使用 upload-area 类，优化拖拽体验
- 🎨 文件列表：使用 file-item 类，优化预览和操作按钮
- 🎨 状态提示：圆角设计 + 图标 + 动画效果
- 🎨 按钮样式：统一的 btn-primary 和 btn-secondary

#### 其他功能页面

- 📝 merge.vue（图像拼接）- 统一样式
- 📝 convert.vue（格式转换）- 统一样式
- 📝 crop.vue（图像裁剪）- 统一样式
- 📝 resize.vue（图像调整）- 统一样式
- 📝 watermark.vue（图像水印）- 统一样式

#### 布局和组件

- 🎨 default.vue: 新增导航栏和页脚
- 🎨 创建 PageHeader 组件，统一页面标题样式
- 🎨 创建全局样式文件 main.css

#### 配置文件

- ⚙️ tailwind.config.js: 添加主色调、阴影、动画配置
- ⚙️ nuxt.config.ts: 引入全局样式文件

### 技术改进

- 📦 更新 Tailwind CSS 配置，添加自定义颜色和阴影
- 📦 创建全局样式类，提高代码复用性
- 📦 优化组件结构，提高可维护性
- 📦 改进响应式设计，支持移动端
