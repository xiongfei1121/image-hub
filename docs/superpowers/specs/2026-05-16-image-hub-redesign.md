# Image Hub 网页设计文档

**日期**: 2026-05-16
**参考网站**: https://www.slimimg.com/zh
**设计风格**: 现代简洁、渐变背景、大量留白

## 一、设计目标

将 Image Hub 从传统工具网站改造为现代简洁的图像处理平台，参考 SlimImg 的设计风格：
- 提升视觉专业度
- 增强用户信任感
- 优化功能展示方式
- 提高转化率

## 二、配色方案

### 主色调
```css
/* 渐变背景 */
--gradient-primary: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
--gradient-hero: linear-gradient(135deg, #f5f7fa 0%, #c3cfe2 100%);

/* 主色 */
--primary-500: #6366f1;  /* Indigo */
--primary-600: #4f46e5;
--primary-700: #4338ca;

/* 辅助色 */
--accent-500: #8b5cf6;  /* Violet */
--accent-600: #7c3aed;

/* 中性色 */
--gray-50: #f9fafb;
--gray-100: #f3f4f6;
--gray-200: #e5e7eb;
--gray-300: #d1d5db;
--gray-400: #9ca3af;
--gray-500: #6b7280;
--gray-600: #4b5563;
--gray-700: #374151;
```

### 应用规则
- 背景：浅灰渐变 `#f5f7fa → #c3cfe2`
- 主按钮：Indigo 渐变
- 卡片：白色 + 轻微阴影
- 文字：深灰 `#1f2937` / 中灰 `#6b7280`

## 三、字体方案

```css
/* 主字体 */
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC',
             'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;

/* 标题字重 */

/* 正文字重 */
```

### 字号规范
- H1: 48px / 3rem（首页标题）
- H2: 32px / 2rem（页面标题）
- H3: 24px / 1.5rem（卡片标题）
- Body: 16px / 1rem（正文）
- Small: 14px / 0.875rem（辅助文字）
